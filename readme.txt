Esse é um código produzido em Python utilizando a versão 3.8.5.

Ele realiza um webscraping (extração de dados de um website) em todos os resultados da primeira página do 
website da amazon, onde realizei a pesquisa "iphone". Após a extração dos dados (dura cerca de 2 minutos), 
os mesmos são armazenados em uma planilha Excel que será criada no mesmo diretório que o arquivo do código 
se encontra.

Bibliotecas utilizadas:
- BeautifulSoup
- requests
- pandas
- lxml
- openpyxl
